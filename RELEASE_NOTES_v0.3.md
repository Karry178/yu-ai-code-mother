### 📋 版本概述

v0.3 版本主要完成了前端用户模块的开发，实现了用户注册、登录、管理等完整功能，与后端 API 完成对接。

---

### ✨ 新增功能

#### 🎨 前端页面开发

**1. 用户认证页面**
- ✅ **用户登录页面**（`UserLoginPage.vue`）
  - 账号密码登录表单
  - 表单验证（账号长度 ≥4，密码长度 ≥8）
  - 登录成功后跳转首页
  - 登录状态持久化（Session）

- ✅ **用户注册页面**（`UserRegisterPage.vue`）
  - 注册表单（账号、密码、确认密码）
  - 密码一致性校验
  - 注册成功自动跳转登录页

**2. 用户管理页面（管理员）**
- ✅ **用户管理页面**（`UserManagePage.vue`）
  - 用户列表展示（表格形式）
  - 分页功能（支持切换页码和每页条数）
  - 用户信息展示（ID、账号、昵称、头像、简介、角色、创建时间）
  - 角色标签显示（管理员/普通用户）
  - 删除用户功能（管理员权限）

**3. 全局组件优化**
- ✅ **导航栏组件**（`GlobalHeader.vue`）
  - 动态菜单（根据权限显示/隐藏管理员菜单）
  - 用户登录状态显示（头像 + 昵称）
  - 下拉菜单（退出登录）
  - 响应式设计

- ✅ **全局布局**（`BasicLayout.vue`）
  - 上中下布局结构
  - 内容区域自适应高度
  - 统一的页面风格

### 🎯 核心功能展示

#### 1. 用户登录流程
```
访问登录页 → 输入账号密码 → 提交登录 → 
后端验证 → 设置 Session → 返回用户信息 → 
前端保存状态 → 跳转首页
```

#### 2. 用户管理流程
```
管理员登录 → 访问用户管理页 → 
后端验证权限 → 返回用户列表 → 
前端展示表格 → 支持分页切换
```

#### 3. 权限控制
- 未登录用户：只能访问首页、登录、注册页
- 普通用户：可访问首页，不能访问管理页面
- 管理员：可访问所有页面

### 🚀 快速开始

#### 前端启动
```bash
cd YuCodeMother-Frontend
npm install
npm run dev
```

#### 访问地址
- 前端：http://localhost:5173
- 首页：http://localhost:5173/
- 登录：http://localhost:5173/user/login
- 注册：http://localhost:5173/user/register
- 用户管理：http://localhost:5173/admin/userManage（需管理员权限）
